Integrasi API (developer friendly)

panduan menghubungkan API Dripsender dengan aplikasi kamu.

Kirim Pesan Text + File (optional)

POST https://api.dripsender.id/send

Gunakan Endpoint ini untuk mengirimkan pesan dari aplikasi kamu via dripsender.id

Request Body

Name
Type
Description

media_url

string

link attachment gambar, file, audio

text

string

isi pesan whatsapp

phone

string

gunakan nomor handphone didahului kode negara tanpa + contoh : 628135199xxxx atau gunakan group id jika kirim ke group, dapatkan group id di menu "groups" dripsender

group_id

string

kosongkan jika bukan kirim ke group, dapatkan group_id melalui menu "groups" di dripsender

api_key*

string

dapatkan api_key di dashboard dripsender.id

OK
